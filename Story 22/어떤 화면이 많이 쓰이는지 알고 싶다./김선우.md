@ 웹로그란?
- 웹 서버에서 어떤 요청을 어떤 사용자가 하는지 남기는 로그

- 아파치나 Nginx, iPlanet과 같은 모든 웹 서버에 공통적으로 제공되는 기능
- 서버에 어떤 사용자가 어떤 요청을 하였고, 결과는 어떠한지 파일에 한 줄씩 쌓아준다.
